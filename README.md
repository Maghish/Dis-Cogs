# Dis-Cogs

> Current version: **v0.0.1**

Discogs is a discord.js bot template that uses the idea of discord.py's Cogs feature and implements it on discord.js

## Requirements:

- Typescript Only
- Supports only `discord.js` v14+

## Features:

- Cogs & Cogs Handler
- Typescript Support
- Simplity
- Extendable
